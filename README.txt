This repository aim to show the different script used to 1) Evaluate and improve the Individual Carbon Weight of Copepods Species (usefull to convert taxonomic abundance data into taxonomic biomass data). 2) Plot the biomass distribution of Copepods functional groups (FG) and trait regionalisation

###____/Data:

All input and output data used in the scripts (See /Data/README.txt). 

PS: The original files downloaded from https://www.st.nmfs.noaa.gov/copepod/search/by_dataset.html are present in '/net/meso/work/svivier/COPEPOD_data/Data_copepod'. To open them all: 'csv_files = glob.glob('/net/meso/work/svivier/COPEPOD_data/Data_copepod/copepod__*/data_src/short-format/*.csv')'

All script used (See /Script/README.txt)
